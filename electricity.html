电学 系统专题

电学综合题的核心是「等效电路 + 约束关系 + 能量视角」。掌握串并联规律与图像语言，可覆盖绝大多数中考与校内压轴结构。

核心公式（理解优先）

I = Q / t
U = IR（纯电阻欧姆定律）
P = UI = I²R = U²/R

焦耳定律：Q = I²Rt；纯电阻电路中电热与电功在数值上常相等，非纯电阻（如电动机）需区分总功与热损耗。

必会能力清单

  • 识别串联「电流相等」、并联「电压相等」的隐含方程
  • 滑动变阻器移动时，局部电阻变化 → 总电阻 → 总电流 → 分压重分配
  • 伏安曲线：斜率与截距的物理意义（注意坐标轴是否互换）
  • 额定功率与实际功率：先求电阻模型，再代入实际电压
  • 家庭电路：火线零线、漏电保护、短路过载的安全分析

知识模块

从元件到系统，从稳态到变化。

电路基础

电源、导线、开关、用电器；通路、断路、短路；电路图与实物图的互译。

电阻与变阻器

电阻定律（长度、横截面积、材料）；滑动变阻器分压与限流接法（初中以限流为主）。

串并联规律

串联 R总=ΣR；并联 1/R总=Σ1/R。比例问题善用「同一电路」约束。

动态电路

局部变化 → 整体电流变化 → 各支路电压电流再分析；表格法与极端法辅助校验。

电功率实验

测小灯泡电功率：接线、量程、灯丝电阻随温度升高而增大导致的非线性。

易错点

混淆电功与电热；忽略电源内阻题干条件；图像不过原点时的截距含义未讨论。
